# Frontend – AI Plagiarism Detection System

This folder contains the Streamlit-based frontend for the plagiarism detection system.

## Responsibilities
- Provide an interactive and user-friendly interface
- Allow users to upload documents
- Display plagiarism summary and detected sources
- Visualize similarity levels using colors and icons
- Show AI-generated content likelihood

## Main File
- `app.py` – Streamlit application

## Features
- Drag-and-drop file upload
- Animated and aesthetic UI
- Color-coded plagiarism indicators
- Real-time communication with backend API

## How It Works
The frontend sends the uploaded document to the FastAPI backend and renders the returned analysis results in a structured, visual format.
